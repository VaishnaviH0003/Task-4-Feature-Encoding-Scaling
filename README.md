# Task 4: Feature Encoding & Scaling

## Dataset Used:
Adult Income Dataset

## Objective:
To preprocess categorical and numerical features using encoding and scaling techniques.

## Steps Performed:
- Removed missing values
- Applied Label Encoding on target column
- Applied One Hot Encoding on categorical features
- Scaled numerical features using StandardScaler
- Saved final processed dataset

## Tools Used:
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Output Files:
- adult_processed.csv
- Jupyter Notebook

## Result:
Dataset is now machine learning ready.
